# week7-d3-userInputApp

Setup global variables for the form
Setup the GET request data from Pokemon API
Setup a render function to display data request on DOM
Setup main to display the returned data from the form when submit is clicked 

 -Two user input fields you can fetch the data from:
 
 -#1 You can enter any id into the Poke ID input field
 -Click submit button
 -GET requested data for that specific ID entered

 -#2 Or you can type the name into the Poke Name input field
 -Click submit button
 -GET requested data for that specific Name entered

 -When one input field has been entered click submit button provided
 -Data will appear on DOM like so (ex. "img", "id", "name", "weight")
